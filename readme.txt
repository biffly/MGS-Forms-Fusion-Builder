== Installation ==
1. Abrir `wp-content/plugins` Folder
2. Crear la carpeta: `MGS-Forms` y colocar el contenifo del ZIP
3. Activar el plugin `MGS-Forms`
4. Ir a `WP-Admin -> MGS-Forms`
